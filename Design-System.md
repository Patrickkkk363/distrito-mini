# Design System - Distrito Mini

## Paleta de Colores

### Colores Primarios (Oscuros y Elegantes)
- **Primary Dark Blue**: `#0F172A` (Slate-900) - Fondo principal, headers
- **Secondary Dark Blue**: `#1E293B` (Slate-800) - Cards, modales, elementos secundarios
- **Accent Blue**: `#3B82F6` (Blue-500) - Botones primarios, enlaces activos, acentos

### Colores Neutros
- **Gray Dark**: `#374151` (Gray-700) - Texto principal
- **Gray Medium**: `#6B7280` (Gray-500) - Texto secundario, placeholders
- **Gray Light**: `#9CA3AF` (Gray-400) - Bordes, divisores
- **White**: `#FFFFFF` - Texto sobre fondos oscuros, fondos de elementos destacados

### Colores de Estado
- **Success**: `#10B981` (Emerald-500) - Confirmaciones, estados positivos
- **Warning**: `#F59E0B` (Amber-500) - Advertencias, stock bajo
- **Error**: `#EF4444` (Red-500) - Errores, cancelaciones
- **Info**: `#06B6D4` (Cyan-500) - Información, notificaciones

### Colores de Categorías (para productos)
- **Electrónica**: `#3B82F6` (Blue-500)
- **Ropa**: `#8B5CF6` (Violet-500)
- **Alimentos**: `#10B981` (Emerald-500)
- **Hogar**: `#F59E0B` (Amber-500)
- **Otros**: `#6B7280` (Gray-500)

## Tipografía

### Fuentes
- **Principal**: Inter (sans-serif moderna)
- **Alternativa**: System UI fonts para compatibilidad

### Escalas
- **H1**: 2.25rem (36px) - Títulos principales
- **H2**: 1.875rem (30px) - Subtítulos
- **H3**: 1.5rem (24px) - Títulos de sección
- **Body Large**: 1.125rem (18px) - Texto principal
- **Body**: 1rem (16px) - Texto estándar
- **Body Small**: 0.875rem (14px) - Texto secundario
- **Caption**: 0.75rem (12px) - Etiquetas, metadata

## Espaciado

### Sistema de Espaciado (basado en 4px)
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)
- **2xl**: 3rem (48px)
- **3xl**: 4rem (64px)

## Componentes Base

### Botones
- **Primary**: Fondo accent blue, texto white, hover más oscuro
- **Secondary**: Borde gray light, texto gray dark, hover fondo gray light
- **Danger**: Fondo error, texto white
- **Padding**: md horizontal, sm vertical
- **Border Radius**: md (6px)

### Cards
- **Background**: Secondary dark blue
- **Border**: 1px gray light
- **Border Radius**: lg (8px)
- **Shadow**: Subtle shadow para profundidad
- **Padding**: lg

### Inputs
- **Background**: Primary dark blue
- **Border**: 1px gray light, focus accent blue
- **Border Radius**: md
- **Padding**: sm

### Navbar
- **Background**: Primary dark blue
- **Text**: White
- **Active Links**: Accent blue
- **Height**: 4rem (64px)

## Tema Oscuro por Defecto

Todo el sistema está diseñado para un tema oscuro elegante, inspirado en aplicaciones modernas como LogiMarket, con excelente legibilidad y jerarquía visual.
